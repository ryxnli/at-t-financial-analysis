# AT&T Financial Analysis

## Overview
This academic project analyzed AT&T's financial statements, market data, and bond information to evaluate the company’s financing structure and estimate its cost of capital.

## Project Objective
The goal of this project was to calculate AT&T’s cost of equity, bond yield/cost of debt, capital structure weights, and weighted average cost of capital (WACC) using Excel.

## Tools Used
- Microsoft Excel
- Financial statement analysis
- WACC calculation
- Bond yield analysis
- Cost of equity calculation

## What I Did
- Organized AT&T balance sheet and income statement data
- Calculated AT&T’s cost of equity using risk-free rate, beta, and market risk premium
- Estimated bond yield/cost of debt using bond pricing inputs
- Calculated book value and market value weights for debt and equity
- Used the cost of debt and cost of equity to estimate WACC

## Skills Demonstrated
- Excel financial modeling
- Corporate finance analysis
- Cost of capital calculation
- Bond yield analysis
- Financial statement organization
